# ChillMeet - Emotional Support Platform

This is a Next.js project built with Tailwind CSS, Framer Motion, and Three.js for a premium, calm, and interactive user experience.

## Features

- **3D Floating Elements**: Uses Three.js via `@react-three/fiber` for a soothing, antigravity aesthetic.
- **Glassmorphism Design**: Premium UI with translucent cards and soft gradients.
- **Framer Motion Animations**: Smooth scrolling and entry animations.
- **Responsive Layout**: Fully responsive design for mobile and desktop.
- **Complete Page Structure**: Includes Home, About, Pricing, FAQ, Become a Listener, and necessary Legal pages.

## Tech Stack

- **Frontend**: Next.js 15 (App Router), React 19
- **styling**: Tailwind CSS v4
- **Animation**: Framer Motion, GSAP (ready)
- **3D**: Three.js, React Three Fiber, React Three Drei
- **Icons**: Lucide React

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages.
- `components/ui/`: Reusable UI components (Navbar, Footer, etc.).
- `components/sections/`: Homepage sections (Hero, HowItWorks, etc.).
- `components/3d/`: Three.js 3D components.
